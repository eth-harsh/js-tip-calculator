# Tip & Bill Calculator

A simple JavaScript program that calculates tip and total bill based on the bill amount.

## Tip Rules
- Bill < ₹500 → 20% tip
- ₹500 to ₹2000 → 15% tip
- Bill > ₹2000 → 10% tip

## How to Run
1. Open `tipCalculator.js` in a browser console, or
2. Run with Node.js: `node tipCalculator.js`

## Concepts Used
- Functions
- If-Else / Else-If
- Truthy/Falsy validation
- Template literals
- Type conversion (Number())
